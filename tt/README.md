# Timetable Bus Transport App

Timetable website for bus transports in Koper.

## Technologies used
 Made with React, Bootstrap, and Node.js

## How to run
 - Run the app.js in the 'Urniki' folder.
        - ```node app.js```
 - Run the React app inside of the aaa
